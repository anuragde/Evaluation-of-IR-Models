1. To run Decision Tree Implementation:
* Run the following command in Command Prompt :
* java -cp DecisionTree.jar   edu.buffalo.decisionTree.ReadFile  "D:\\Sem3\\601DM\\project3_dataset4.txt"
2. To run Random Forest Decision Tree Implementation:
* Run the following command in Command Prompt :
* java -cp DecisionTree.jar edu.buffalo.RandomForestDT.ReadFileRandomForest "D:\\Sem3\\601DM\\project3_dataset1.txt" 10
(i.e. <dataset file location> <number of trees>)
3. To run Boosting Decision Tree Implementation:
* Run the following command in Command Prompt :
* java -cp DecisionTree.jar edu.buffalo.boostingDT.ReadFileBoosting "D:\\Sem3\\601DM\\project3_dataset1.txt" 10
(i.e. <dataset file location> <number of trees>)
